# 📰 News Aggregator Platform (daily.dev Style)

A **modern news aggregator platform** where developers and tech enthusiasts can **discover, bookmark, and share articles** from trusted sources.  
Built with **Node.js, Express, MongoDB, and React**, this project demonstrates **scalable full-stack development**, API integration, and a clean reading experience.

## ✨ Key Features

- 🔑 **User Authentication** – Secure login and personalized feed.  
- 🌍 **News Feed** – Aggregates articles from RSS feeds & APIs (e.g., TechCrunch, Dev.to).  
- 🔖 **Bookmarks** – Save articles to read later.  
- 👍 **Reactions & Comments** – Engage with the community by liking and discussing articles.  
- 📊 **Analytics** – Track trending posts, sources, and user activity.  
- 🎨 **Responsive Frontend** – Clean UI optimized for reading on any device.  

---

## 🛠️ Tech Stack

- **Frontend:** React / Vanilla JS + CSS  
- **Backend:** Node.js + Express (Go optional)  
- **Database:** MongoDB (Mongoose ODM)  
- **Integration:** RSS parsing + Open Graph scraping  

---

## 🚀 What I Focused On
